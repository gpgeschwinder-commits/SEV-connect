# SEV CONNECT — Installations-Anleitung
## Schritt-für-Schritt ohne technisches Wissen

---

## WAS IHR BRAUCHT
- Einen Computer (Windows oder Mac)
- Eine E-Mail-Adresse
- Ca. 30-45 Minuten Zeit

---

## SCHRITT 1 — GitHub Konto erstellen

1. Öffnet euren Browser (Chrome empfohlen)
2. Geht auf: **github.com**
3. Klickt oben rechts auf "Sign up"
4. Gebt eure E-Mail ein → Passwort wählen → Benutzername: "sevconnect"
5. Wählt den KOSTENLOSEN Plan
6. Bestätigt eure E-Mail (prüft euer Postfach)

---

## SCHRITT 2 — Neues Projekt anlegen

1. Nach dem Login: Klickt oben rechts auf das "+" Symbol
2. Klickt auf "New repository"
3. Repository name: **sev-connect**
4. Wählt "Public"
5. Haken setzen bei "Add a README file"
6. Klickt auf den grünen Button "Create repository"

---

## SCHRITT 3 — Dateien hochladen

Ihr habt von uns folgende Dateien erhalten (als ZIP):
- src/App.jsx
- src/index.js
- public/index.html
- public/manifest.json
- package.json

So ladet ihr sie hoch:
1. In eurem neuen Repository klickt auf "Add file"
2. Wählt "Upload files"
3. Zieht ALLE Dateien in das Upload-Fenster
4. Schreibt unten: "Erste Version SEV Connect"
5. Klickt auf "Commit changes"

---

## SCHRITT 4 — Vercel Konto erstellen

1. Geht auf: **vercel.com**
2. Klickt auf "Sign Up"
3. Wählt "Continue with GitHub"
4. Erlaubt Vercel den Zugriff auf GitHub

---

## SCHRITT 5 — App auf Vercel deployen

1. Nach dem Login klickt auf "Add New Project"
2. Sucht euer "sev-connect" Repository → klickt "Import"
3. Alles so lassen wie es ist
4. Klickt auf den orangen Button "Deploy"
5. Wartet ca. 2 Minuten
6. ✅ Fertig! Vercel zeigt euch eure URL:
   z.B. **sev-connect.vercel.app**

---

## SCHRITT 6 — App auf dem Handy installieren

### Android (mit Chrome):
1. Chrome öffnen
2. URL eingeben: sev-connect.vercel.app
3. Rechts oben: Drei Punkte (⋮) tippen
4. "Zum Startbildschirm hinzufügen" wählen
5. "Hinzufügen" bestätigen
6. ✅ App erscheint auf dem Homescreen

### iPhone (mit Safari):
1. Safari öffnen (NICHT Chrome!)
2. URL eingeben: sev-connect.vercel.app
3. Unten: Teilen-Symbol tippen (Quadrat mit Pfeil)
4. "Zum Home-Bildschirm" wählen
5. "Hinzufügen" bestätigen
6. ✅ App erscheint auf dem Homescreen

---

## SCHRITT 7 — Test

1. App auf dem Handy öffnen
2. Sprache wählen
3. "Ich bin Fahrer" tippen
4. Fahrer auswählen → Anmelden
5. Alles durchklicken und testen

---
